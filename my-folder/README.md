# Smart Inventory and Sales Management System

## Project Overview

The Smart Inventory and Sales Management System is a console-based Python application developed using Object-Oriented Programming (OOP).

It helps small businesses manage:

- Products
- Inventory
- Suppliers
- Customers
- Sales
- Purchases
- Billing
- Reports

---

## Features

### Authentication

- Admin Login
- Username & Password Verification
- Maximum 3 Login Attempts

## How to Run

1. Install Python 3.

2. Open the project in VS Code.

3. Open Terminal.

4. Run

```
python main.py
```

or

```
python3 main.py
```
